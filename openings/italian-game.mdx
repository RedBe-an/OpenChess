# Italian Game

Italian Game (1. e4 e5 2. Nf3 Nc6 3. Bc4)는 가장 오래된 체스 오프닝 중 하나입니다.

## 개요

이탈리아 게임은 16세기부터 알려진 오프닝으로, 중앙 제어와 빠른 개발을 추구합니다.

## 주요 아이디어

- 킹사이드 비숍을 활성화
- f7 약점 공격 준비
- 빠른 캐슬링을 통한 킹 안전성 확보

## 주요 변형

### 클래식 Italian Game

```chess
1. e4 e5
2. Nf3 Nc6
3. Bc4 Bc5
4. c3 Nf6
5. d3
```

### Hungarian Defense

```chess
1. e4 e5
2. Nf3 Nc6
3. Bc4 Be7
```

이 오프닝은 초보자와 고수 모두에게 인기가 있으며, 전략적 이해를 높이는 데 도움이 됩니다.

# Sicilian Defense

Sicilian Defense (1. e4 c5)는 1. e4에 대한 가장 인기 있는 응답 중 하나입니다.

## 개요

시칠리아 수비는 비대칭적인 게임을 만들어 복잡하고 전략적인 포지션을 추구합니다.

## 주요 아이디어

- 중앙에서의 비대칭적 구조
- 흑이 d6-d5 또는 e6-e5 push를 준비
- 퀸사이드에서의 반격 기회

## 주요 변형

### Najdorf Variation
```chess
1. e4 c5
2. Nf3 d6
3. d4 cxd4
4. Nxd4 Nf6
5. Nc3 a6
```

### Dragon Variation
```chess
1. e4 c5
2. Nf3 d6
3. d4 cxd4
4. Nxd4 Nf6
5. Nc3 g6
```

### Accelerated Dragon
```chess
1. e4 c5
2. Nf3 g6
3. d4 cxd4
4. Nxd4 Bg7
```

시칠리아 수비는 매우 복잡하며 깊은 이론적 지식이 필요합니다.
